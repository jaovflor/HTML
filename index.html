Elementos de Texto

A tag <strong> para deixar o texto em negrito.

A tag <em> serve para deixar o texto em itálico.

A tag <u> serve para sublinhar o texto.

A tag <strike> serve para riscar o texto.

A tag <mark> serve para marcar o texto.

A tag <small> serve para reduzir o tamanho do texto.

A tag <ins> serve para representar um texto inserido.

A tag <br> serve para quebrar linhas.
